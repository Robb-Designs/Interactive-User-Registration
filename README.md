# Interactive Registration Form

A client-side registration form that provides real-time validation feedback for user inputs. This project focuses on form validation, user experience, and clean separation of logic using helper functions.

---



---

## How It Works

### HTML Structure

The form includes the following fields:

- Username
- Email
- Password
- Confirm Password

Each field has a corresponding error message container.



---

Each field has its own validation function:

```js
validateUsername()
validateEmail()
validatePassword()
validateConfirmPassword()
